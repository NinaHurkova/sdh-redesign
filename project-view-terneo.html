Background

Terneo™ is an innovative Ukrainian company that provides state-of-the-art home automation and heating solutions. Terneo solutions are designed to be efficient and environmentally friendly, helping users reduce energy consumption and lower their carbon footprint.

Objective

To develop a cloud solution for home automation. The main objective was to create a unified ecosystem for connecting and managing thermostats via smartphone Wi-Fi. The system should enable users to remotely control Terneo thermostats, which regulate heating devices in the home (electric underfloor heating, convection heaters, radiators, etc.).

Solution

Our team developed a unified IoT ecosystem for terneo™ devices management. We made the heating system control entirely remote and user-friendly. Wi-Fi Internet connection provides quick access to the settings of all thermostats listed in a user account. Setting a smart home became a lot more convenient since the management system access is provided both via a website and mobile apps (Android, iOS). All the data is gathered, processed, and stored securely in the cloud.

Heating charts

The app displays the dynamics of heating activity per day, week, month, or year. A user can optimize the heating schedule by analyzing past periods' data.

Heating conditions setup

Users can schedule a heating turn on and off at the appointed time and days. Terneo heating systems will maintain the set temperature for a planned duration.

Heating zoning

Users can add up to 16 heating scenarios for a specific location per day depending on a preferable temperature at a particular time of day and night.

Intuitive interface

All the devices connected to the app are labeled with different colors. A device in an energy-efficient mode is marked as green and becomes yellow and orange in a heating mode.

Wi-Fi thermostats management

A single app allows remote managing of all Terneo thermostats that regulate heat appliances at home (electric floor heating, convection heater, radiators, etc.).

Cloud data storage

Cloud solution offers exceptional securely stored and transmitted, ensuring privacy and protection.

Geofencing

This GPS-perimeter-based technology for heating automation via IFTTT allows users to interact with the system via GPS-enabled smartphones based on where they are.
